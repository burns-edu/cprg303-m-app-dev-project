## Setup Instructions:

### Prerequisites

- Node.js
- Expo Go app (Android) or Android Studio

### Installation

1. Clone the repository
   git clone https://github.com/burns-edu/cprg303-m-app-dev-project.git
2. Install dependencies
   npm install
3. Create a `.env` file in the project root with the following:
   EXPO_PUBLIC_SUPABASE_URL=your-project-url
   EXPO_PUBLIC_SUPABASE_ANON_KEY=your-anon-key
4. Start the app
   npx expo start
